## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/shonuff1221/ejm-landing-page-main.git
```

2. Go into project

```bash
cd ejm-landing-page
```

3. Install dependencies

```bash
pnpm install
```

4. Run project

```bash
pnpm dev
```
